# CRM-Invoice-Platform-Analysis
Power BI dashboard comparing invoice platforms based on features, usability, and business requirements.
## Project Overview
This project analyzes and compares different invoice platforms to identify the most suitable solution based on business requirements.

## Objectives
- Compare invoice platform features
- Analyze usability and integration capabilities
- Provide insights for selecting the best platform

## Tools Used
- Power BI
- Microsoft Excel
- Data Analysis
- Data Visualization

## Dashboard Features
- Feature comparison analysis
- Platform scoring
- User experience evaluation
- Business insights

## Skills Demonstrated
- Data cleaning
- Data modeling
- Dashboard design
- Business analysis
- Data storytelling
